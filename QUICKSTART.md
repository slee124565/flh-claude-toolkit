# QUICKSTART

這份文件給第一次使用 `flh-claude-toolkit` 的同仁。

目標只有一個：

- 把你的 Mac 準備成可以開始用 Claude 協作工作的環境

## 建議順序

1. 熟悉 Terminal 基本操作
2. 安裝 Homebrew
3. 安裝 Node.js
4. 安裝 GitHub `gh` CLI
5. 安裝 Claude Code CLI
6. 設定 terminal / MCP
7. 視需要安裝 Google Workspace `gws`
8. 完成後再到 `llm-wiki-kb`

## 完成後做什麼

當你已經可以：

- 在 Terminal 執行基本指令
- 使用 Claude Code CLI
- 讓 Claude Desktop 看到需要的工具

就可以前往：

- [llm-wiki-kb](../llm-wiki-kb/README.md)

在那裡開始把工作知識整理進知識庫。
