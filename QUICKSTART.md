# QUICKSTART（macOS）

這份文件給第一次使用 `flh-claude-toolkit` 的 Mac 同仁。

Windows 用戶請看：[windows-quickstart.md](windows-quickstart.md)

目標只有一個：

- 把你的 Mac 準備成公司同事可用的 Claude 工作環境

## 必要安裝順序

1. 熟悉 Terminal 基本操作
2. 確認 `git` CLI 可用
3. 安裝 Homebrew
4. 安裝 Node.js
5. 安裝 Claude Desktop App
6. 安裝 Claude Code CLI
7. 安裝 Google Workspace `gws`
8. 設定 terminal / MCP
9. 完成後再到 `llm-wiki-kb`

## 進階選配

如果你後面需要 private repo、PR、issue 或其他 GitHub 進階操作，再安裝 GitHub `gh` CLI。

## 完成後做什麼

當你已經可以：

- 在 Terminal 執行基本指令
- 使用 `git` clone public repo
- 打開 Claude Desktop App
- 使用 Claude Code CLI
- 使用 `gws`
- 讓 Claude Desktop 看到需要的工具

就可以前往：

- [llm-wiki-kb](https://github.com/slee124565/llm-wiki-kb/blob/main/README.md)
