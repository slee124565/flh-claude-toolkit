# Windows QUICKSTART

這份文件給第一次使用 `flh-claude-toolkit` 的 Windows 同仁。

Mac 用戶請看：[QUICKSTART.md](QUICKSTART.md)

目標只有一個：

- 把你的 Windows 電腦準備成公司同事可用的 Claude 工作環境

## 必要安裝順序

1. 熟悉 PowerShell 基本操作
2. 確認 `winget` 可用
3. 確認 `git` CLI 可用
4. 安裝 Node.js
5. 安裝 Claude Desktop App
6. 完成後再到 `llm-wiki-kb`

## 進階選配

如果你後面需要以下能力，再補裝對應工具：

- 如果你需要用 Claude Code CLI 協助工作：安裝 Claude Code CLI
- 如果你想把 Claude Code 接進 Claude Desktop：設定 terminal / MCP
- 如果你使用 `Claude Code` TUI，且需要讀 Gmail / Drive / Calendar：安裝 Google Workspace `gws`
- 如果你主要在 Claude Desktop GUI 編輯 Google Sheets：安裝 Google Sheets MCP
- 想做 private repo、PR、issue 或其他 GitHub 進階操作：安裝 GitHub `gh` CLI
